## ðŸ†˜ Troubleshooting

### Problem: Can't login

**Solution:**
1. Click "Forgot Password"
2. Enter your email
3. Check email for reset link
4. Click link and create new password
5. Login with new password

### Problem: Application won't submit

**Reasons:**
- Missing required fields
- Email format wrong
- File too large (when upload available)

**Solution:**
1. Check all fields have values
2. Make sure email is valid (someone@example.com)
3. Refresh page
4. Try again

### Problem: Can't find a job

**Solution:**
1. Use search bar (type keywords)
2. Try different keywords ("engineer" vs "developer")
3. Remove filters (they might be too strict)
4. Check again tomorrow (new jobs posted daily)
5. Save jobs for later (don't need to apply immediately)

### Problem: Status not updating

**Solution:**
1. Refresh page (F5)
2. Logout and login again
3. Wait 30 seconds (sometimes API is slow)
4. Contact HR if still not updated

### Problem: Email notifications not received

**Reasons:**
- Check spam/junk folder
- Email address misspelled in profile
- Notifications disabled in settings

**Solution:**
1. Check profile â†’ Email settings
2. Make sure notifications are enabled
3. Add career portal to email contacts
4. Check spam folder

### Problem: Forgot my password

**Solution:**
```
1. Click "Forgot Password"
2. Enter email address
3. Check inbox for reset email
4. Click "Reset Password" link
5. Create new password
6. Login with new password
```

### Problem: Can't create account

**Reasons:**
- Email already used
- Password doesn't meet requirements
- Required fields missing

**Solution:**
1. Try login instead (you might already have account)
2. Use password with 8+ characters, uppercase, number
3. Fill all required fields (marked with *)
4. Use real email address
